# Food Specials Gallery Images

This directory is for storing food special images that will be displayed in the Food Specials Gallery.

## How to Add Images

1. Add your image files to this directory (or to `/public/pics/`)
2. In the admin panel, go to Food Specials
3. When creating or editing a food special, enter the image path in the "Image Path" field
   - Example: `/food-specials/your-image.jpg`
   - Or: `/pics/your-image.jpg`
4. The image will automatically appear in the gallery at `/food-specials-gallery`

## Supported Formats

- JPG/JPEG
- PNG
- WebP

## Image Recommendations

- Recommended aspect ratio: 4:3
- Recommended size: 800x600px or larger
- Keep file sizes reasonable for fast loading
