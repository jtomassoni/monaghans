# Creating and Managing Drink Specials

Promote daily or weekly beverage deals, including happy hour.

## Create Special

1. Go to [Drink Specials](/admin/drink-specials)
2. Click **"New Drink Special"**
3. Fill details and save

## Special Details

### Required
- **Title**: Special name (e.g., "Happy Hour", "Wine Wednesday")
- **Type**: "Drink" (auto-set)

### Optional
- **Description**: Special details
- **Price Notes**: Pricing (e.g., "$2", "Half Price")
- **Weekday Selection**: Days of week
- **Time Window**: Time range (e.g., "4pm-6pm", "All Day")
- **Date Range**: Start/end dates
- **Status**: Active (visible) or Inactive (hidden)

## Weekday-Based Specials

Specials that repeat weekly:

1. Check days when special applies:
   - Monday through Sunday
2. Special appears on those days each week

**Example**: "Happy Hour" Monday-Friday 4pm-6pm:
- Days: Monday, Tuesday, Wednesday, Thursday, Friday
- Time Window: "4pm-6pm"

## Date Range Specials

Specials for specific date ranges:

1. Set **Start Date**
2. Optionally set **End Date**
3. Special appears on all dates in range

**Example**: "Holiday Drink Special" Dec 15-31:
- Start: December 15
- End: December 31

## Time Windows

When during the day the special applies:

Enter time range in **Time Window**:
- "4pm-6pm" (Happy Hour)
- "9pm-Close" (Late Night)
- "All Day"
- "11am-2pm" (Lunch)

## Common Types

### Happy Hour
- Typically: Monday-Friday, 4pm-6pm
- Examples: "$2 Beers", "$5 Cocktails", "Half Price Wine"

### Daily Specials
- Examples: "Wine Wednesday", "Thirsty Thursday", "$1 Shot Night"

### All Day Specials
- Examples: "$3 Drafts All Day", "Happy Hour All Day Sunday"

## Edit Special

1. Go to [Drink Specials](/admin/drink-specials)
2. Click the special
3. Update details
4. Save

## Delete Special

1. Go to [Drink Specials](/admin/drink-specials)
2. Click delete icon (trash)
3. Confirm

## Status

- **Active**: Visible on calendar and homepage
- **Inactive**: Hidden (not deleted)

Use inactive to temporarily hide without deleting.

## Display Locations

Drink specials appear on:
- **Calendar** (scheduled dates)
- **Homepage** (when active)
- **Promotional materials** (if integrated)

## Tips

- Make pricing clear and easy to understand
- Use consistent time windows
- Plan weekly recurring specials
- Feature popular drinks
- Keep specials current
