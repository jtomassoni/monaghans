# Creating One-Time Events

Create single events that don't repeat.

## How to Create

### From Calendar
1. Go to **Calendar** (`/admin`)
2. Click the date
3. Select **"New Event"**
4. Fill details and save

### From Events List
1. Go to **Calendar**
2. Switch to **List View**
3. Click **"New Event"**
4. Fill details and save

### Quick Actions
1. Open **Quick Actions** menu (any admin page)
2. Select **"New Event"**
3. Fill details and save

## Event Details

### Required
- **Title**: Event name (e.g., "Live Music Night")
- **Start Date & Time**: When it begins
- **End Date & Time**: When it ends (skip for all-day events)

### Optional
- **Description**: Event details
- **Venue Area**: Location (e.g., "Main Bar", "Patio")
- **Tags**: Categories (e.g., "music", "sports")
- **All Day**: Check if event lasts all day
- **Image**: Upload event image
- **Status**: Active (visible) or Inactive (hidden)

## Status

- **Active**: Visible on calendar and website
- **Inactive**: Hidden (useful for drafts)

## Tips

- Use tags to organize similar events
- Add venue area so customers know where to go
- Include important details in description (cover charges, age restrictions)
- Upload images to make events stand out
