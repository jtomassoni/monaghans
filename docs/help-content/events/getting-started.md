# Getting Started with Calendar & Events

Manage all events, specials, and announcements in one place.

## Overview

Visual view of scheduled content:
- **Events**: One-time or recurring (live music, trivia, etc.)
- **Specials**: Daily or weekly food and drink specials
- **Announcements**: Time-sensitive with publish/expiry dates

## Create Your First Event

1. Go to [Calendar](/admin) page in admin
2. Click **"Create Event"** (or click a date)
3. Fill details:
   - **Title**: Event name
   - **Date & Time**: When it occurs
   - **Description**: Event details
   - **Venue Area**: Location (optional)
   - **Tags**: Categories (optional)
4. Click **"Save"**

## Calendar Views

View calendar in different ways:
- **Week View**: Current week
- **Month View**: Current month
- Use navigation arrows to move between weeks/months

## Quick Tips

- **Drag and Drop**: Drag events to reschedule
- **Recurring Events**: Create events that repeat daily, weekly, or monthly
- **Color Coding**: Events color-coded by type
