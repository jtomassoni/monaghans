# Business Settings and Information

Manage business info, hours, and system configuration.

## Access Settings

1. Go to **Settings** (`/admin/settings`)
2. Review and update options

## Business Information

Update business details:

### Basic Info
- **Business Name**: Establishment name
- **Address**: Physical address
- **Phone**: Contact number
- **Email**: Contact email
- **Website**: Website URL

### Social Media
- **Facebook**: Facebook page URL
- **Instagram**: Instagram profile URL
- **Other Social Links**: Additional profiles

## Business Hours

Set operating hours:

1. Find **Business Hours** in Settings
2. Set hours for each day:
   - Monday through Sunday
   - Opening and closing times
3. Mark days as **Closed** if not operating
4. Save

### Tips
- Keep hours current, especially holidays
- Note special hours for holidays/events
- Hours display in local time zone
- Set hours for each location if multiple

## Calendar Hours

Set hours displayed on calendar:

1. Find **Calendar Hours** in Settings
2. Set **Start Hour**: When day view starts (e.g., 8 AM)
3. Set **End Hour**: When day view ends (e.g., 11 PM)
4. Save

**Note**: Controls visible time range on calendar, not business hours.

## Feature Flags

Control which features are enabled:

1. Find **Feature Flags** (Admin only)
2. Toggle features:
   - **Online Ordering**: Enable/disable
   - **Staff Management**: Enable/disable
   - **Other Features**: Additional toggles
3. Changes take effect immediately

**Note**: Typically visible to admin users only.

## Homepage Settings

Customize homepage (see [Customizing Homepage](../homepage/customizing-homepage.md)):
- Hero section
- About content
- Images

## System Settings

Additional configuration:
- **Timezone**: Business timezone
- **Date Format**: Date display format
- **Time Format**: 12-hour or 24-hour

## Saving Changes

1. Make changes in Settings
2. Click **"Save"** or **"Update"**
3. Changes save immediately
4. Some changes may require page refresh

## Tips

- Keep business info current
- Update hours for holidays/special events
- Test changes on public site
- Keep backup of important settings
