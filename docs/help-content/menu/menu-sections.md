# Managing Menu Sections

Organize menu items into logical categories.

## What are Menu Sections?

Categories that group related items. Examples:
- Appetizers
- Entrees
- Desserts
- Drinks
- Specials

## Create Section

1. Go to **Menu** (`/admin/menu`)
2. Click **"Sections"** tab
3. Click **"New Section"** (or "+")
4. Fill details:
   - **Name**: Section name (e.g., "Appetizers")
   - **Menu Type**: Food or Drink
   - **Display Order**: Order on menu (lower = first)
5. Save

## Edit Section

1. Go to **Menu** → **Sections**
2. Click section name or edit icon
3. Update details
4. Save

## Delete Section

**Important**: Can only delete sections with no items.

1. Go to **Menu** → **Sections**
2. Click delete icon (trash)
3. Confirm

**Note**: If section has items, either:
- Delete or move all items first, OR
- Move all items to another section

## Reorder Sections

**Display Order** controls section order:

1. Edit section
2. Change **Display Order** number
3. Lower numbers appear first (1, 2, 3...)
4. Save

**Tip**: Use increments of 10 (10, 20, 30...) for easy insertion later.

## Menu Types

Assign sections to menu types:
- **Food Menu**: Food items
- **Drink Menu**: Beverage items

Helps organize when you have separate food and drink menus.

## Best Practices

- Keep it simple: 5-10 sections usually enough
- Logical grouping: Group items customers expect together
- Clear names: Use names customers understand
- Consistent ordering: Use display order for structure

## Common Section Names
