# Setting Menu Item Availability

Temporarily hide items without deleting. Useful for out of stock, seasonal, or testing.

## Understanding Availability

- **Available**: Visible on menu, can be ordered
- **Unavailable**: Hidden from menu (not deleted)

## Set Availability

### Quick Toggle
1. Go to **Menu** → **Items**
2. Find the item
3. Click **Status Toggle** (Available/Unavailable)
4. Updates immediately

### When Editing
1. Edit the menu item
2. Find **"Available"** checkbox/toggle
3. Check/uncheck
4. Save

## When to Use Unavailable

- **Out of Stock**: Temporarily out of ingredient
- **Seasonal**: Only available certain times
- **Testing**: Testing before full launch
- **Temporary Removal**: Might bring back later

## Availability vs. Deletion

**Using Availability (Recommended)**:
- ✅ Data preserved
- ✅ Easy to reactivate
- ✅ Maintains history
- ✅ No need to recreate

**Deleting Items**:
- ❌ Permanently removed
- ❌ Must recreate if needed
- ❌ Loses history and data

## Bulk Updates

Update multiple items:
1. Go to **Menu** → **Items**
2. Use status toggle on each item
3. Changes save automatically

## Indicators

In menu management:
- **Green/Active**: Available
- **Gray/Inactive**: Unavailable
- Status badges show current state

## Customer View

- **Available**: Appear on public menu
- **Unavailable**: Hidden from customers
- No indication shown for temporarily unavailable items

## Tips

- Review unavailable items periodically
- Add notes about why items are unavailable
- Use status toggle for quick changes
- Plan ahead for seasonal changes
