# Digital Signage and Specials TV

Display specials, events, and announcements on TV screens throughout your establishment.

## What is Digital Signage?

Displays content on TV screens or monitors:
- **Specials TV**: Food and drink specials
- **Events Display**: Upcoming events
- **Announcements**: Time-sensitive announcements
- **Slideshow**: Rotates through content types

## Access Settings

1. Go to [Digital Signage](/admin/signage)
2. Configure displays

## Set Up Specials TV

### Basic Setup

1. Go to [Digital Signage](/admin/signage)
2. Find **Specials TV** settings
3. Configure:
   - **Display URL**: URL to open on TV/monitor
   - **Auto-refresh**: Enable automatic updates
   - **Slide Duration**: How long each slide shows

### Display URL

URL to open on TV or monitor:
- Typically: `https://yoursite.com/specials-tv`
- Page auto-refreshes and displays content
- Bookmark URL on TV's browser

## Content Display

Signage automatically shows:
- **Active Specials**: Current food and drink specials
- **Upcoming Events**: Today and upcoming events
- **Announcements**: Published within date range

## Slideshow Settings

Configure content rotation:

1. **Slide Duration**: How long each slide shows (default: 10 seconds)
2. **Slide Transition Duration**: Time for slide transition animation (default: 0.4 seconds). Uses TV-friendly slide transitions for smooth performance on large displays.
3. **Content Order**: Order content types appear

## Wall-Mounted Display Setup

### Hardware Requirements

- TV or monitor (any size)
- Device with browser (smart TV, computer, tablet, streaming device)
- Internet connection

### Setup Steps

1. **Connect Device**: Connect TV/monitor to device with browser
2. **Open Display URL**: Navigate to specials TV URL
3. **Enter Fullscreen**: Press F11 (computers) or use fullscreen mode
4. **Auto-Refresh**: Page automatically updates content
5. **Position**: Mount/position where customers can see

### Recommended Setup

- **Smart TV**: Use built-in browser
- **Computer/Tablet**: Connect via HDMI or wirelessly
- **Streaming Device**: Device with browser (Chromecast, Fire TV, etc.)

## Managing Content

Content comes from:
- **Specials**: Active food and drink specials
- **Events**: Calendar events
- **Announcements**: Published announcements

To update:
1. Update specials, events, or announcements in admin
2. Changes automatically appear (with auto-refresh)

## Auto-Refresh

Signage automatically refreshes:
- **Refresh Interval**: Typically every 30-60 seconds
- **No Manual Updates**: Updates automatically
- **Always Current**: Shows most up-to-date info

## Troubleshooting

### Display Not Showing Content
- Check specials/events/announcements are **Active**
- Verify display URL is correct
- Ensure internet connection working

### Content Not Updating
- Check auto-refresh enabled
- Verify content is Active in admin
- Clear browser cache on display device

### Display Looks Wrong
- Check screen resolution settings
- Ensure browser in fullscreen mode
- Verify content formatting in admin

## Tips

- Place displays where customers can easily see
- Regularly update specials and announcements
- Check displays periodically
- Have backup device for technical issues
- Use high-quality images and clear text
